# Array dimensions

##

|  data     | = (rank) | ≠ (length) | ≡ (depth) | ≢ (shape) | note                 |
|-----------|---------:|-----------:|----------:|:---------:|----------------------|
|   `1`     |        0 |          1 |         0 |      `⟨⟩` | a base data type     |
|  `< 1`    |        0 |          1 |         1 |      `⟨⟩` | a unit array         |
| `7‿8‿9`   |        1 |          3 |         1 |     `⟨3⟩` | a list, 1D array     |
|`< 7‿8‿9`  |        0 |          3 |         2 |      `⟨⟩` | a unit array of list |
